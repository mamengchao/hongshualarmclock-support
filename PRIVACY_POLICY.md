# 隐私政策

本隐私政策说明了「红薯闹钟」（以下简称"本应用"）如何收集、使用和保护您的个人信息。

**最后更新日期：2026 年 5 月 16 日**

请您在使用本应用前仔细阅读本隐私政策。一旦您开始使用本应用，即表示您已理解并同意本隐私政策的全部内容。

---

## 1. 信息收集

本应用仅收集以下必要信息：

- **闹钟数据**：您设置的闹钟名称、时间、响铃模式、重复规则等，这些数据仅存储在您的设备本地，不会上传至任何服务器。
- **设备权限**：本应用需要获取通知权限以在设定的时间提醒您。该权限仅用于本地闹钟提醒功能。
- **广告标识符**：本应用集成了第三方广告服务（中国大陆广告服务商），在关闭闹钟页面展示 Banner 广告以及用户选择观看激励视频广告时，广告 SDK 可能会收集设备的广告标识符（IDFA）、设备类型、操作系统版本等非个人信息，用于提供广告并统计广告效果。这些数据由广告 SDK 独立处理，开发者不会主动收集或存储。您可以在系统设置中限制广告追踪（设置 → 隐私 → 跟踪）。
- **友盟 SDK 数据**：本应用集成友盟+ SDK（简称「友盟SDK」），用于统计分析服务，以了解应用的使用情况和优化用户体验。友盟SDK 可能会收集以下信息：设备信息（Android ID/IDFA/IDFV/OAID/OpenUDID/GUID；可选-IMEI/IMSI/ICCID）、网络信息、位置信息（可选）、应用列表（可选）。详细内容请参见友盟隐私政策。

## 2. 信息存储

- 所有闹钟数据均存储在您设备的本地 SQLite 数据库中，我们不会将任何数据传输到外部服务器。
- 应用内置的关闭闹钟所需数据（包括但不限于诗词、数学题、文字等）从应用安装包中的资源文件读取，无需联网。
- 广告标识符等信息由广告 SDK 临时处理，开发者不存储这些信息。

## 3. 权限说明

本应用可能请求以下权限：

- **通知权限**：用于在闹钟触发时向您发送提醒。您可以在系统设置中随时开启或关闭此权限。

## 4. 第三方服务

4.1 本应用集成了第三方广告 SDK（中国大陆广告服务商）。在关闭闹钟页面，应用会展示 Banner 横幅广告；用户也可选择观看激励视频广告来获取答案并关闭闹钟。广告 SDK 可能会自动收集匿名化的设备信息（如广告标识符、设备型号、操作系统版本等），并遵循其隐私政策。

4.2 本应用集成友盟+ SDK 用于统计分析。友盟SDK 可能会收集设备信息（Android ID/IDFA/IDFV/OAID/OpenUDID/GUID；可选-IMEI/IMSI/ICCID）、网络信息、位置信息（可选）、应用列表（可选），用于统计应用使用数据和分析用户行为。详细内容请参见[友盟隐私政策](https://www.umeng.com/page/policy)。

4.3 本应用本身不主动收集、上传或存储用户的个人信息。所有闹钟数据仅存储在设备本地。应用仅在关闭闹钟页面展示 Banner 广告和激励视频广告时与广告服务器建立必要的网络连接，不会以其他任何理由联网。

4.4 您可以通过设置界面重置广告标识符或关闭个性化广告功能（系统设置 → 隐私 → 跟踪 → 关闭「允许 App 请求跟踪」）。关闭后您仍会看到广告，但广告的相关性会降低。

## 5. 数据安全

5.1 由于所有闹钟数据均存储在设备本地，不存在数据传输过程中的泄露风险。我们建议您妥善保管自己的设备，避免设备丢失或被他人未经授权使用。

5.2 对于广告 SDK 在广告展示过程中收集的信息，由广告服务提供商按照其自身的安全标准和隐私政策负责保护，开发者不承担因此产生的数据安全责任。

## 6. 儿童隐私

6.1 本应用面向一般大众用户，非专门面向儿童设计。本应用不会故意收集未满 14 周岁的未成年人的个人信息。

6.2 如果监护人发现未成年人在未获其同意的情况下使用本应用并向我们提供了个人信息，请通过本政策底部的联系方式与我们联系，我们将在合理期限内删除相关信息。

## 7. 用户权利与数据控制

7.1 由于所有闹钟数据仅存储在您的设备本地，您可以通过应用内的编辑或删除功能直接管理这些数据。卸载本应用将自动清除设备上的所有闹钟数据。

7.2 对于广告 SDK 收集的数据，您可以通过以下方式行使控制权：

- 关闭个性化广告：系统设置 → 隐私 → 跟踪 → 关闭「允许 App 请求跟踪」
- 重置广告标识符：系统设置 → 隐私 → 跟踪 → 点击「还原广告标识符」

## 8. 政策更新

本隐私政策可能不时更新。重大变更将在应用内显著位置通知。建议您定期查阅本隐私政策。

## 9. 联系我们

如您对本隐私政策有任何疑问、意见或请求（包括查阅、更正、删除您的个人信息），请通过应用内反馈渠道与我们联系。我们将在合理期限内回复。
