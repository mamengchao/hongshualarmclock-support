# 红薯闹钟 — 技术支持

感谢使用红薯闹钟！

## 常见问题

**Q: 闹钟没有按时响？**
A: 请检查以下设置：
- 系统通知权限是否已开启（设置 → 红薯闹钟 → 通知 → 允许通知）
- 设备是否处于静音模式或勿扰模式
- 低电量模式可能影响后台提醒
- 建议在重要事项前使用系统自带闹钟作为备用提醒

**Q: 答题无法关闭闹钟？**
A: 如果长时间无法答出题目，可以选择观看广告视频来关闭闹钟。

**Q: 节假日跳过了不该跳过的日期？**
A: 节假日数据来自应用内置的节假日配置文件，如有偏差可以反馈给我们更新。

**Q: 如何删除所有闹钟数据？**
A: 卸载应用即可自动清除所有本地数据。
